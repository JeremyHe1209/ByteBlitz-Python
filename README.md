## 码卡：一款充满 OI 元素的卡牌游戏

**码卡** 是一款富含 **OI**（信息学奥赛）元素的卡牌游戏，由两位中学生开发。这个项目正在积极开发中。

**想了解我们的进度？**

下载 [game.tldr](https://github.com/JeremyHe1209/ByteBlitz/edit/blob/game.tldr) 并使用 [tldraw](https://www.tldraw.com) 打开。这个文件详细展示了每个类的具体成员、类之间的关系以及我们目前的完成情况。
